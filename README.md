# Projeto-Phoenix-Desktop-Windows
Agente Autônomo Multi-Funcional para Windows (Desktop/Console) com automação de UI, voz (TTS/STT), banco de conhecimento (Programação/IA/Engenharia de Prompt), e auto-melhoria contínua via Gemini Agent Mode. Versão desktop do Project Phoenix com suporte a .NET/C#, WPF, e processamento local avançado.
